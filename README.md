## 🇧🇷 Português

### 📖 Sobre o Projeto

**Tronus** é um sistema gamificado de apoio acadêmico desenvolvido para ajudar alunos calouros que estão com dificuldades no semestre. A ideia é transformar a jornada acadêmica em uma aventura épica, onde cada matéria é um **Reino** a ser conquistado.

### 🎮 Como Funciona

| Elemento | Descrição |
|---|---|
| 👑 **Plebeu** | Título inicial do calouro |
| ⚔️ **Reinos** | Representam as matérias do semestre |
| 🏰 **Fases** | Cada reino possui 3 fases para conquistar |
| ⭐ **Estrelas** | Cada fase vale até 3 estrelas |
| 🎖️ **Cavaleiro** | Conquistado ao atingir 15 estrelas |
| 🦅 **Duque** | Conquistado ao atingir 30 estrelas |
| 👑 **Rei** | Conquistado ao atingir 50 estrelas — todos os reinos dominados |
| 🏛️ **Imperador** | Papel do veterano que orienta e apoia o calouro |

### ✨ Funcionalidades

- 🔐 Cadastro e login com nome de usuário ou email
- 📱 Recuperação de senha via número de celular
- 🧙 Criação de personagem com escolha de gênero e avatar
- 🗺️ Sistema de Reinos (matérias) com fases e estrelas
- 🏆 Progressão de títulos conforme conquistas
- 💾 Banco de dados SQLite local via `sql.js`

### 🚀 Como Rodar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/Pim-Unip2025/Tronus.git
```
2. Abra a pasta no **VSCode**
3. Instale a extensão **Live Server**
4. Clique com o botão direito no `index.html` → **Open with Live Server**
5. Pronto! O projeto abrirá no navegador 🎉

### 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura das páginas
- **CSS3** — Estilização e animações
- **JavaScript** — Lógica do sistema
- **sql.js** — Banco de dados SQLite rodando no navegador
- **Boxicons** — Ícones
- **Google Fonts** — Tipografia (Poppins)

## 🇺🇸 English

### 📖 About the Project

**Tronus** is a gamified academic support system developed to help freshman students who are struggling during the semester. The idea is to transform the academic journey into an epic adventure, where each subject is a **Kingdom** to be conquered.

### 🎮 How It Works

| Element | Description |
|---|---|
| 👑 **Plebeian** | Freshman's initial title |
| ⚔️ **Kingdoms** | Represent the semester's subjects |
| 🏰 **Phases** | Each kingdom has 3 phases to complete |
| ⭐ **Stars** | Each phase is worth up to 3 stars |
| 🎖️ **Knight** | Achieved at 15 stars |
| 🦅 **Duke** | Achieved at 30 stars |
| 👑 **King** | Achieved at 50 stars — all kingdoms conquered |
| 🏛️ **Emperor** | Role of the veteran who guides and supports the freshman |

### ✨ Features

- 🔐 Register and login with username or email
- 📱 Password recovery via phone number
- 🧙 Character creation with gender and avatar selection
- 🗺️ Kingdom system (subjects) with phases and stars
- 🏆 Title progression based on achievements
- 💾 Local SQLite database via `sql.js`

### 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/Pim-Unip2025/Tronus.git
```
2. Open the folder in **VSCode**
3. Install the **Live Server** extension
4. Right-click `index.html` → **Open with Live Server**
5. Done! The project will open in your browser 🎉

### 🛠️ Tech Stack

- **HTML5** — Page structure
- **CSS3** — Styling and animations
- **JavaScript** — System logic
- **sql.js** — SQLite database running in the browser
- **Boxicons** — Icons
- **Google Fonts** — Typography (Poppins)
